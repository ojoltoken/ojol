Download TokenPocket atau wallet lain. Pastikan ada OJOL Token di wallet kamu.
2. Konek Wallet
Klik "Mulai Staking" dan sambungkan wallet-mu ke aplikasi. Hanya 1 klik!
3. Pilih Durasi
Tentukan jumlah OJOL yang mau di-stake dan durasi (30/90/180/360 hari).

Klik Stake dan mulai dapatkan reward harian. Semakin besar stake, semakin besar reward!
